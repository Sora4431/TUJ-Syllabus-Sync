## Setup

- npm install

- Create .env and put your Google Client ID there.

- Make sure http://localhost:3000 is in your Google Cloud Console's Authorized JavaScript origins.

---

## Run

- npm run dev

---

## Logic

Google GIS works in the browser without a backend.

Google Calendar API can be called from the frontend using the token.
